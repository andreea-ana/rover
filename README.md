Un program Python care:

- elimină duplicatele
- păstrează doar numerele pare
- grupează numerele în 3 categorii (small, medium, large)
- calculează suma pentru fiecare categorie

## Exemplu

Input:
[5, 2, 8, 2, 3, 8, 10, 5, 60, 44, 12, 60]

Output:
{
  'small': [2, 8],
  'medium': [10, 12, 44],
  'large': [60]
}

Sume:
small: 10
medium: 66
large: 60
